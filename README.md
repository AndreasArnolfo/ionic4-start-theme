Ionic 4 Start Theme
==========


Requirements
------------

* Node 10+
* Npm 6+
* Ionic 4+
* Ionic Lab 1+

```
$ brew update && brew upgrade
$ brew install node
$ npm install -g npm@latest
$ npm install -g ionic@rc
$ npm install -g @ionic/lab@rc semver
```

Created with Ionic CLI version 4.0.0 RC.9:

```
$ npm install
$ ionic serve -l
```
